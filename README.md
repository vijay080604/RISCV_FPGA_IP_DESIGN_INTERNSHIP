# RISC-V FPGA IP Design Internship

This repository documents my journey through the **RISC-V FPGA IP Design Internship** conducted by **VLSI System Design (VSD)**.

The internship provides hands-on experience in RISC-V software development, FPGA-based system design, and custom IP development by combining practical implementation with engineering concepts.

Throughout this internship, I will document each task with implementation details, execution results, architecture diagrams, and the key concepts learned during the development process.

---

## Internship Objectives

- Set up the complete RISC-V development environment.
- Understand the RISC-V software development workflow.
- Learn cross-compilation and ISA simulation.
- Design and integrate custom hardware IPs.
- Implement and validate designs on FPGA.
- Build a strong foundation in RISC-V and digital system design.
